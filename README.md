# Community Savings Groups Digital Management System (CSG-DMS)

A web-based platform for managing community savings groups in Rwanda.

## How to Run

1. **Clone the repository**
   ```bash
   git clone https://github.com/62Laura/CSG-DMS-Frontend.git
   cd CSG-DMS-Frontend
   ```

2. **Navigate to frontend directory**
   ```bash
   cd frontend
   ```

3. **Install dependencies**
   ```bash
   npm install
   ```

4. **Start the application**
   ```bash
   npm start
   ```

The application will open automatically at `http://localhost:3000`
